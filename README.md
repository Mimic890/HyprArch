# HyprArch
# Arch Linux with Hyprland
### С этой инструкцией по установке вы сможете с полного нуля установить у себя оригинальный Arch Linux с Hyprland'ом.

После запуска ОРИГИНАЛЬНОГО образа с официального сайта: https://archlinux.org/releng/releases/

### 1.1 Сначала для дальнейшей установки нужно подключится к сети (инструкция для беспроводного подключения):
```
systemctl start iwd
iwctl
station list
station wlan0 scan
station get-networks
station wlan0 connect 'network_name'
quit
```
#### Для проверки подключения:
`ping google.com`
### 1.2 Для упрощенной установки на русском языке нужно установить шрифт, поддерживающий русский:
`setfont cyr-sun16`
### 2.1 Запускаем скрипт упрощенной установки:
`archinstall`
### Здесь настраиваем все на свой выбор
### 2.2 В конце выбираем `Установить`, дожидаемся окончания установки, после этого выбираем использовать chroot (да или нет) и перезагружаем компьютер командой `reboot`
### 3.1 После запуска вы попадёте на экран приветствия, в левом верхнем углу в пункте `Сеанс` выберите `Hyprland`, введите пароль и вы попадете в Arch linux Hyprland
#### 4.1 Сперва нужно настроить конфиг файл на первое время
#### 4.2 Для открытия командной строки нажмите `SUPER+Q` и с помощь текстового редактора заходим в конфиг hyprland'а 
`vim .config/hypr/hyprland.conf` переходим в режим редактирования клавишей A, перед строчкой `autogenerated...` ставим `# `, чтобы сохранить изменения нажмите `ESC ⇒ и напишите :w`
#### 4.3 Чтобы добавить русский язык найдите и измеите чтоб было так `kb_layout = us,ru` `kb_options = grp:win_space_toggle`
#### 4.4 Для настройки дисплея в конфиге введите `hyprctl monitors` в самом конфиге записываем так: `monitor = интерфейс, разрешение@герцовка, позиция, маштаб (в моем случяе: monitor=eDP-1,1920x1080@60,0x0,1)`
#### 4.5 Чтобы подключится к интернету в самой операционной системе пропишем эти команды:
```
nmcli
nmcli device wifi list
nmcli device wifi connect network_name password network_password
```
#### Теперь нужно установить пакетный менеджер yay для установки репозиториев комьюнити
```
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
```
#### Установим нужные шрифты `sudo pacman -S ttf-font-awesome otf-font-awesome ttf-jetbrains-mono`
