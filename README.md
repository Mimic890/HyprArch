# HyprArch
# Arch Linux with Hyprland
### С этой инструкцией по установке вы сможете с полного нуля установить у себя оригинальный Arch Linux с Hyprland'ом.

После запуска ОРИГИНАЛЬНОГО образа с официального сайта: https://archlinux.org/releng/releases/

### 1.1 Сначала для дальнейшей установки нужно подключится к сети (инструкция для беспроводного подключения):
`systemctl start iwd`

`iwctl`

`station list`

`station wlan0 scan`

`station get-networks`

`station wlan0 connect 'network_name'`

`quit`
#### Для проверки подключения:
`ping google.com`
### 1.2 Для упрощенной установки на русском языке нужно установить шрифт, поддерживающий русский:
`setfont cyr-sun16`
### 2.1 Запускаем скрипт упрощенной установки:
`archinstall`
### 2.2 В пункте `Archinstall language` выбираем `Russian`
### 2.3 Я пропущу пункт `зеркала`, потому что по умолчанию всё и так хорошо работает
### 2.4 В пункте `Локализация` выбираем `Язык локали` (для поиска нажмите `/`) и ищем `ru_RU.UTF-8`
### 2.5 Возвращаемся назад и выбираем `конфигурацию диска ⇒ Разбивка на разделы` и выбираем один из данных предложенных вариантов (я выбираю `Использовать оптимальную схему разделов по умолчанию` и нужный диск, на котором будет сама операционная система, также я выберу файловую систему `ext4` и еще нужно выбрать, будет ли `отдельный раздел для /home`, я выбираю `да`)
### 2.6 Возвращаемся назад, я пропущу пункт `Шифрование диска`
### 2.7 В следующем пункте `Загрузчик` я выберу `Grub`
### 2.8 В пункте `Подкачка` я оставлю по умолчанию
### 2.9 Пункт `Имя хоста` — это название компьютера
### 2.10 `Пароль root` вводим два раза
### 2.11 В пункте `Учетная запись пользователя` добавляем пользователя, вводим имя и пароль (я выберу такой же пароль, как и пароль root), потом `Должен ли пользователь быть суперпользователем`(я выберу `да`), подтверждаем и выходим
### 2.12 Наконец в пункте `Профиль ⇒ Тип ⇒ Desktop ⇒ Hyprland ⇒ polkit/seatd (я выберу первое) ⇒ Графический драйвер (я оставлю по умолчанию) ⇒ Экран приветствия (я выберу sddm)`
### 2.13 `Звуковой сервер` выбирайте сами
### 2.14 `Ядра` я оставлю linux
### 2.15 В пункте `Дополнительные пакеты` я запишу `git firefox waybar hyprpaper hyprlock`
### 2.16 В пункте `Настройка сети` я выберу `Использовать NetworkManager`
### 2.17 В пункте `Часовой пояс` (для поиска нажмите `/`) и ищем `Europe/Moscow`
### 2.18 `Автоматическая синхронизация времени` я оставлю `True`
### 2.19 `Дополнительные репозитории` я выберу `multilib`
### 2.20 И наконец-то выбираем `Установить` (после этого нужно нажать ENTER) и дожидаемся окончания установки, после этого выбираем использовать chroot (да или нет), пишем в командную строку `exit` и перезагружаемся в новую операционную систему командой `reboot`
### 3.1 После запуска вы попадёте на экран приветствия, в левом верхнем углу в пункте `Сеанс` выберите `Hyprland`, введите пароль и вы наконец то попадете в Arch linux Hyprland
#### 4.1 Сперва нужно настроить конфиг файл на первое время
#### 4.2 Для открытия командной строки нажмите `SUPER+Q` и с помощь текстового редактора заходим в конфиг hyprland'а 
`vim .config/hypr/hyprland.conf` переходим в режим редактирования клавишей A, перед строчкой `autogenerated...` ставим `# `, чтобы сохранить изменения нажмите `ESC ⇒ и напишите :w`
#### 4.3 Чтобы добавить русский язык найдите и измеите чтоб было так `kb_layout = us,ru` `kb_options = grp:win_space_toggle`
